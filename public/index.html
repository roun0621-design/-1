실시간 대회 결과 확인

P-R : Node를 통해 육상 대회의 실시간 경기 결과를 확인할 수 있습니다. 대회를 선택한 후 대시보드에서 종목별 진행 상황과 결과를 바로 확인하세요.

주요 기능

  • 대시보드 — 전 종목의 진행 상황을 한눈에 확인, 영상 링크 바로가기
  • 실시간 결과 — 경기가 끝나는 즉시 기록과 순위 확인 가능
  • 종합 순위 — 여러 조로 진행되는 종목도 통합 순위로 확인
  • 실시간 현황 — 소집 상태, 진행중인 경기 확인
  • 소집실 운영 — 바코드/배번 스캔, 릴레이 팀 단위 체크인
  • 기록 입력 — 트랙/필드/도로/복합 종목별 기록 관리
  • 관리자 패널 — 선수/종목 관리, 조편성, 엑셀 업로드, 데이터 백업

즐겨찾기

대시보드에서 관심 종목 옆의 별 아이콘을 눌러 즐겨찾기에 추가하면, 해당 종목이 상단에 고정되어 빠르게 확인할 수 있습니다.

운영 매뉴얼

P-R : Node 대회 운영 가이드

1. 접속 및 로그인

  • 운영키: 소집실 운영, 기록 입력 등 일반 운영 기능에 접근
  • 관리자키: 모든 운영 기능 + 대회/종목/선수 관리, 시스템 설정
  • 심판별 개별 운영키를 생성하면 로그에 심판명이 자동 기록됩니다

2. 대회 흐름

대회 생성 → 엑셀 업로드(선수/종목) → 소집실 운영 → 기록 입력 → 결과 확인

3. 소집실 운영 (Callroom)

  • 좌측에서 종목 선택 → 우측에서 선수 체크인
  • 바코드 스캐너 또는 배번(BIB) 직접 입력으로 체크인
  • 릴레이 종목: 팀 단위로 표시, 클릭하면 출전 선수 명단 확인 가능
  • 전원 소집 확인 후 "소집 완료" 버튼 → 경기 상태가 '진행중'으로 전환
  • 불참(No-show) 처리도 가능, 선수 추가 버튼으로 추가 등록

4. 기록 입력 (Record)

  • 트랙 종목: 시간을 초 단위로 입력 (예: 10.52)
  • 도로/단체 종목: H:MM:SS 형식 (예: 1:38:36), 소수점 없이 표시
  • 필드 거리 종목: 각 시기별 거리(m) 입력, 파울은 빈칸
  • 필드 높이 종목: 바 높이 설정 후 O/X/Pass 입력
  • 복합 종목: 서브이벤트별 기록 입력 → WA 점수 자동 계산
  • 비고란: 기록 저장 후 비고를 입력해도 기록이 유지됩니다
  • 소집이 완료되지 않은 종목도 도로/단체 종목은 바로 기록 입력 가능

5. 결과 확인 (Results)

  • 전체 종목의 결과를 조별/통합순위로 확인
  • 여러 조로 진행된 종목은 "종합순위" 탭으로 전체 통합 확인
  • Excel, PDF, PNG로 내보내기 가능

6. 관리자 기능 (Admin)

  • 선수 관리: 추가/수정/삭제, BIB/바코드 관리
  • 종목 관리: 추가/수정/삭제, 자동정렬, 영상 URL 설정
  • 조편성: 종목 클릭 → 하단 패널에서 조 추가/삭제, 선수 이동, 스몰넘버(레인) 직접 편집
  • 엑셀 업로드: 연맹 명단 엑셀 업로드 → 선수, 종목, 조 자동 생성
  • 경기 완료/되돌리기: 라운드 완료 처리 (관리자키 필요 시 되돌리기)
  • 대회 삭제: 관리자키 인증 후 대회 전체 데이터 삭제 가능
  • 데이터 백업: JSON 형식으로 전체 대회 데이터 다운로드

7. 실시간 업데이트 (SSE)

모든 페이지는 Server-Sent Events로 실시간 업데이트됩니다. 기록 입력, 소집 완료, 경기 완료 등의 변경사항이 모든 접속 기기에 즉시 반영됩니다.

8. 문제 해결

  • 데이터가 안 보이면 페이지 새로고침 시도
  • SSE 연결이 끊어지면 자동 재연결 (우측 하단 표시)
  • 데이터 백업: 관리자 → 대회 설정 탭 → JSON 백업 다운로드
